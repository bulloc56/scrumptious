WELCOME TO PMT

A Project Management Tool for Software Developers

What is it?

PMT is a project management tool primarily for software developers. It incoporates the functionality of a kanban board, which allows the user to organize their tasks as cards that can be moved to match specific project deadlines (sprints), and a chat system for each specific project.




PMT

A Project Management Tool for Software Developers

What is it?

PMT gives software developers a packaged workflow tool. PMT combines a chat system and a Kanban board to allow for convenient communications between team members during Scrum Sprints. Similar to other websites, members will need an account and will require an invitation to join projects that they did not make themselves.

Why use it?

Planning and managing your project with PMT will help your team hit deadlines, stay on focused, and make awesome products. By organizing and planning out Sprints, large projects feel small. Confusion becomes awareness. And work gets done!
